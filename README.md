
# GoLang Zero to Hero Guide

Welcome! This repository is dedicated to learning Go from scratch, with a structured chapter-based approach. The goal is to progress from beginner to advanced Go developer by investing 1-2 hours daily.

## Learning Plan

### Chapters

1. **Introduction to Go**
	- What is Go?
	- Setting up your environment
	- First Go program

2. **Basic Syntax & Data Types**
	- Variables, constants
	- Data types: int, float, string, bool
	- Operators

3. **Control Structures**
	- If, else, switch
	- Loops: for, range

4. **Functions**
	- Defining and calling functions
	- Parameters and return values
	- Variadic functions

5. **Collections**
	- Arrays, slices
	- Maps

6. **Structs & Interfaces**
	- Structs and methods
	- Interfaces and polymorphism

7. **Error Handling**
	- Error types
	- Custom errors

8. **Concurrency**
	- Goroutines
	- Channels
	- Select statement

9. **Modules & Packages**
	- Creating and using packages
	- Dependency management

10. **Testing**
	 - Writing tests
	 - Using Goâ€™s testing tools

11. **Building Real Projects**
	 - CLI tools
	 - Web servers (basic REST API)

## Daily Commitment

- **Time Investment:** 1-2 hours per day
- **Approach:** Read, code, and experiment with examples for each chapter.
- **Progress Tracking:** Update this README with your progress and notes.
